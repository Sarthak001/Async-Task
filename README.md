## Prerequisite

> Nodejs & npm

## Running the development server:

```bash

git clone https://github.com/Sarthak001/Async-Task.git
cd Async-Task
npm install
npm run dev

```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
